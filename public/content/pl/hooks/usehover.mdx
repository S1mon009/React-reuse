## Opis

Hook `useHover` wykrywa, kiedy element jest najechany kursorem. Zwraca logiczny stan hovered, który wskazuje, czy mysz znajduje się obecnie nad elementem, oraz ref do podłączenia do elementu, który chcemy śledzić.

## API

- `hovered: boolean` - `true`, jeśli element jest aktualnie najechany, `false` w przeciwnym razie.
- `ref: React.RefObject<T>` - `Ref`, który należy podłączyć do śledzonego elementu w celu wykrywania stanu najechania.
