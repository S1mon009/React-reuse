## Description

The `If` component enables conditional rendering in React. It accepts three props: `condition`, `children`, `else`. This provides a clean, semantic way to express an “if…else” flow in JSX without cluttering your code with ternary operators.

## Parameters

- `condition: boolean` – a boolean expression that determines which block to display.
- `children: ReactNode` – the content to render when `condition` is true.
- `else?: ReactNode` – (optional) the content to render when `condition` is false.
