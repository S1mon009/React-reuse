## Description

The `Each` component is a utility function designed to simplify array rendering in React. It takes an array (`of`) and a render function, which is invoked for each item in the array. The render function receives two arguments: the current item (`item`) and its index (`index`), returning a corresponding JSX element. By abstracting the iteration logic, Each provides a clean, reusable, and type-safe way to render lists dynamically.

## Parameters

- `render(item: T, index: number): JSX.Element` - A required function that defines how to render each item in the array. It takes two arguments (`item` - The current element from the array, `index` - The index of the current element).
- `of: any[]` - A required array of items that will be iterated over. The render function is called on each element of this array.
