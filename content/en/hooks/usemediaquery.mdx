## Description

The `useMediaQuery` hook enables you to apply responsive design techniques by listening to CSS media query changes. It returns a boolean indicating whether the media query matches the current viewport, allowing you to conditionally render components or apply styles based on the query.

## API

- `isMobile: boolean` - `true` if the media query matches the current viewport, otherwise `false`.

## Parameters

- `query: string` - A string representing the media query you want to listen to.
