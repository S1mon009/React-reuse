## Description

The `useHover` hook detects when an element is hovered over. It returns a hovered boolean state that indicates whether the mouse is currently over the element and a ref to be attached to the element you want to track.

## API

- `hovered: boolean` - `true` if the element is currently being hovered over, `false` otherwise.
- `ref: React.RefObject<T>` - A `ref` that needs to be attached to the target element for detecting hover state.
