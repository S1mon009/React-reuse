## Description

The `Show` component in React enables conditional rendering of child components based on specified conditions. It uses two subcomponents: `Show.When` and `Show.Else`. The `Show.When` component renders its children if the `isTrue` prop is true, while `Show.Else` renders if no valid condition is met. This component simplifies conditional logic by embedding it directly within JSX.
